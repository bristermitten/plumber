StaticModule

(JVM) @Target([AnnotationTarget.CLASS]) annotation class StaticModule

Only applicable to subclasses of Aspect.

Used to configure a static module for an Aspect. This is useful when an Aspect requires some bindings from Guice to be injected into it.

Normally, injection would happen, and then Aspect.getModule would be installed, but no bindings would exist from the module when the Aspect is created.

This annotation solves that problem. If an Aspect has it, the given target will be installed first, and then the Aspect will be injected with the bindings from the module.
