EventController

(JVM) interface EventController<T> : Listener where T : PlayerEvent, T : Cancellable

A wrapper for handling events that provides simple boilerplate handling Currently, only supports Cancellable instances of PlayerEvent All instances have an EventPriority of EventPriority.NORMAL For performance reasons, if no handling functionality is defined, the controller is not registered into Bukkit
