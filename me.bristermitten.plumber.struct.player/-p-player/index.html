PPlayer

(JVM) interface PPlayer : KeyHolder, Extendable<PPlayer, PlayerExtension>

Plumber wrapper for the Player class Allows lots of boilerplate on Players do be done easily

Instances can be obtained from PPlayerManager if necessary, otherwise it's advised to use events or command injection, rather than storing and manually obtaining instances.

Per online player, one PPlayer instance exists. As with Bukkit's Player, this will be removed from storage if the player logs out, so it's advised not to store instances if possible, as they will not be able to be garbage collected otherwise. If necessary, weak references are advised
